# 数据卡

## NHANES 2003-2004 及相关周期

- 数据来源：CDC / NCHS NHANES
- 数据类型：公开结构化临床与体检变量
- 核心任务：糖尿病足相关高危因素识别与综合风险分层
- 样本粒度：受试者级
- 关键字段方向：人口学、糖尿病相关指标、周围神经病变、ABI、既往病史
- 是否公开：是
- 使用限制：需遵循 NHANES 数据使用规则
- 当前用途：主线风险提示原型

## DFUC Challenge 数据

- 数据来源：DFUC challenge / Medical Image Analysis challenge report
- 数据类型：足溃疡图像与分割标签
- 核心任务：足溃疡区域分割与图像异常演示
- 样本粒度：图像级
- 关键字段方向：图像路径、掩膜标签、任务划分
- 是否公开：受限公开
- 使用限制：按许可协议申请
- 当前用途：扩展图像模块

## STANDUP 及相关热成像研究数据

- 数据来源：研究型公开数据与相关论文
- 数据类型：足底 RGB 与热成像
- 核心任务：热特征分析、分类研究、接口预留
- 样本粒度：图像级 / 受试者级
- 关键字段方向：模态类型、采集条件、样本类别、热特征
- 是否公开：研究型公开或有限开放
- 使用限制：预处理流程较特定，需要单独适配
- 当前用途：热成像扩展接口

## 指南与宣教知识资料

- 数据来源：IDF、CDC、ADA、综述与公开教育资料
- 数据类型：文本证据
- 核心任务：知识图谱构建、问答、护理宣教
- 样本粒度：证据条目级
- 关键字段方向：实体、关系、来源、年份、证据类型
- 是否公开：大多可公开访问
- 使用限制：需规范引用与人工筛选
- 当前用途：主线知识图谱问答

## 中医护理与外治知识

- 数据来源：教材、综述、规范性资料
- 数据类型：宣教知识条目
- 核心任务：护理宣教与辅助建议
- 样本粒度：知识条目级
- 关键字段方向：适用场景、禁忌提示、建议类型
- 是否公开：视来源而定
- 使用限制：需人工筛选并避免夸大疗效
- 当前用途：主线护理宣教模块
