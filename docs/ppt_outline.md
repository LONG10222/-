# PPT 页级大纲

## 第 1 页：项目标题与团队分工

- 标题：基于公共数据与知识图谱的糖尿病足风险提示与中医护理宣教智能体
- 小标题：公共数据 + 知识图谱 + Streamlit 原型
- 成员分工：需求分析 / 数据建模 / 网页演示

## 第 2 页：需求分析

- 糖尿病人群规模大
- 足部并发症高危因素识别重要
- 居家管理与基层风险提示辅助存在明显缺口

## 第 3 页：范围收口与主线定义

- 主线：NHANES 风险提示 + 知识图谱问答 + 网页 + 综合报告
- 扩展 1：DFUC 图像演示
- 扩展 2：热成像与传感器接口预留
- 当前主线不是临床确诊预测，而是综合风险分层

## 第 4 页：公开数据来源

- NHANES：结构化临床与高危因素
- DFUC：足部溃疡图像分割，许可申请后使用
- STANDUP/热成像研究数据：扩展接口来源
- 指南/综述：IDF、CDC、ADA 等知识证据来源

## 第 5 页：数据资源组织方案

- 多源异构数据资源库
- 统一字段 schema
- 数据卡与字段映射表
- 分任务 benchmark 与证据图谱

## 第 6 页：知识图谱设计

- 实体类型
- 关系类型
- 样例三元组
- 证据来源字段

## 第 7 页：系统总流程图

- 用户输入：基本信息 / 症状问卷 / 可选足部图像
- 数据处理层：结构化风险评分 / 图谱检索 / 可选图像分析
- 智能体协调层：结果整合 / 风险解释 / 护理宣教生成
- 输出层：风险等级 / 高危因素解释 / 就医建议 / 护理宣教 / 综合报告

## 第 8 页：功能演示与硬件方案

- 风险问卷页
- 知识图谱问答页
- 综合报告页
- 硬件分层：当前版本 / 升级版本

## 第 9 页：评估指标与应用效果

- V1：规则覆盖率、解释完整度、响应时间、问答命中率
- V2：AUC、F1、Recall、Dice/IoU 等
- 应用效果：风险提示、护理宣教、辅助转诊、报告输出

## 第 10 页：总结

- 主线清晰
- 数据可得
- 系统可做
- 答辩友好、可持续扩展
